# liucuier
liucuier blog

blog 地址：http://www.liucuier.com
目前正在策划中，敬请期待

### 安装依赖
```shell
npm install -g yog2
cd base
npm install
# base/app.js 内可以更改运行端口
```
参考文档：(YOG2)[http://fex.baidu.com/yog2/docs/]

### 启动项目

cd abse

yog2 run

### 进行开发以及编译
- 终端窗口1：
```shell
cd home
sh dev.sh d
# 重看 dev.sh 脚本，参数 d 是时时编译，不用重启项目，参数 rr 可以实现往所需服务器上传代码，各取所需
```

- 终端窗口2：
```shell
cd common
sh dev.sh d
```
